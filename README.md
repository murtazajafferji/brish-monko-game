# Brish & Marko Adventure

A small browser game made from the original drawings.

## Heroes
- **Brish** — punch really hard
  - `J` = Mega Punch
  - `K` = Ground Smash
- **Marko** — sharp shoes + magic power
  - `J` = Sharp Shoe Dash
  - `K` = Magic Flower projectile

## Levels
- **Lego Land**
- **Beach World**
- **Mini World**

## Run it
From this folder:

```bash
python3 -m http.server 8000
```

Then open:

<http://localhost:8000>

## Controls
- Move: Arrow keys or `A` / `D`
- Jump: `W`, `Up`, or `Space`
- Power 1: `J`
- Power 2: `K`
- Restart: `R`
